# Tic-Tac-Toe-using-C
A lightweight, interactive Tic-Tac-Toe game built to demonstrate clean logic and smooth user interaction.
🕹️ Game Overview
This is a 2-player game played directly in the command prompt or terminal. The program manages the game state using a 2D character array and provides a clean, text-based interface for coordinate input.

✨ Key Features
Grid Rendering: Dynamic updates to the 3x3 board after every move.

Input Validation: Prevents players from choosing occupied cells or entering invalid coordinates.

Condition Logic: Smart detection for horizontal, vertical, and diagonal wins.

Draw Detection: Automatically identifies when no further moves are possible.
